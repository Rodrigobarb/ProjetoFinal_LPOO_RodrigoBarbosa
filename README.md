## 📋 Descrição do Projeto

Sistema completo de gerenciamento de torneios de e-sports desenvolvido em Python com interface gráfica Tkinter e persistência em PostgreSQL. O sistema permite cadastro de jogos, equipes, jogadores, criação de torneios com diferentes formatos, inscrições, geração automática de chaveamento e registro de resultados.
